# Diabetes Detection using AI

This project demonstrates diabetes detection using machine learning with a simple web interface.

## Features

- Data preprocessing and visualization (Jupyter notebook)
- Model training (Logistic Regression)
- Model evaluation (Accuracy, Confusion Matrix, Classification Report)
- **Web interface for quick predictions (Streamlit)**

## How to Run

### 1. Install dependencies
```
pip install -r requirements.txt
```

### 2. Run the web app
```
streamlit run app.py
```
Open the provided local URL in your browser.

### 3. (Optional) Explore the notebook
```
jupyter notebook diabetes_detection.ipynb
```

## Data

The app and notebook use the Pima Indians Diabetes Dataset, downloaded automatically.

## Notes

- The web tool is for demonstration and learning only.
- Python 3.7+ is required.
